# Contributing to TaskManager-CRUD

First off, thank you for taking the time to contribute. Any help that improves this project — bug fixes, features, documentation, or suggestions — is appreciated.

## How Can You Contribute?

### 1) Reporting Bugs

If you find a bug:

* Go to the Issues section of the repository
* Create a new issue
* Clearly describe:

  * What happened
  * What you expected to happen
  * Steps to reproduce the issue
  * Screenshots (if applicable)
  * Device/OS details (Android/iOS version, emulator/real device)

### 2) Suggesting Features

Feature suggestions are welcome.

When suggesting a feature:

* Explain the problem it solves
* Describe the expected behavior
* Keep it practical and relevant to a CRUD task management app

### 3) Code Contributions

To contribute code:

1. Fork the repository
2. Clone your fork:

   ```
   git clone https://github.com/your-username/TaskManager-CRUD.git
   ```
3. Create a new branch:

   ```
   git checkout -b feature-name
   ```
4. Make your changes
5. Commit clearly:

   ```
   git commit -m "Add: short description of change"
   ```
6. Push your branch:

   ```
   git push origin feature-name
   ```
7. Create a Pull Request to the main repository

## Contribution Guidelines

* Write clean, readable code
* Follow existing folder structure and naming patterns
* Avoid unnecessary dependencies
* Test your changes before submitting
* Keep pull requests focused (one feature/fix at a time)

## Code Style

* Maintain consistent formatting
* Use meaningful variable and function names
* Comment only where necessary (avoid obvious comments)

## Pull Request Checklist

Before submitting a PR, make sure:

* Code runs without errors
* No existing functionality is broken
* Changes are relevant to the project
* Commit message is clear and concise

## Questions?

If you're unsure about something, open an issue first and discuss before starting major work.

Thank you for helping improve this project.
