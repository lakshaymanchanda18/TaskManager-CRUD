### 4) Can I use the app without internet?

Yes. The application works completely offline since tasks are stored locally on the device.

### 5) Where are my tasks stored?

All tasks are saved locally on your device using local storage. No data is sent to external servers.

### 6) Is my data secure?

Yes. Task data remains on your device and is not transmitted anywhere. Basic local protection is provided, but this is a learning project and not meant for handling sensitive information.

### 7) How are my tasks backed up?

Tasks are stored locally. For backup, you can extend the project by integrating cloud storage or exporting data manually.

### 8) How do I run this project locally?

1. Clone the repository

   ```
   git clone https://github.com/your-username/TaskManager-CRUD.git
   ```
2. Move into the project folder

   ```
   cd TaskManager-CRUD
   ```
3. Install dependencies

   ```
   npm install
   ```
4. Run the app

   ```
   npx react-native run-android
   ```
